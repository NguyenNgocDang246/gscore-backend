# GScore Backend

## Requirements

- Node.js 18+ (Node.js 20+ recommended)
- MongoDB

## Installation

1) Install dependencies:

```bash
npm install
```

2) Create `.env` from `.env.example` and update values

## Run the app

Start dev server:

```bash
npm run start:dev
```

The server runs at:

```
http://localhost:4000
```
